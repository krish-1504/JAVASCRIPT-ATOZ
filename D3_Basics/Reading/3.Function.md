#Function Syntax :
```
function name(parameter1, parameter2, parameter3) {
  // code to be executed
  return x;
}
```


## Why Function :
With function you can reuse Code


## Constructor :

```
const fun = new Function("a","b","return a*b");
```

## Arrow Function (ES6):
```
const hello = (a,b) =>{
    return a+b;
}
console.log(hello(10,20));
```
